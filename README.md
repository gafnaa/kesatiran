# Kesatiran

Website "Kesatiran" merupakan proyek web yang dibuat dengan menggunakan **Next.js** dan **Tailwind CSS**.

- [x] Setup Tailwind CSS v4

- [x] Home (`/`) – pengantar dan landing page
